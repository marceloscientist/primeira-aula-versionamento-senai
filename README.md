# Git Learning Project

## Overview

This is a basic exploratory Git project created by SENAI tutor **Marcelo de Carvalho Santana** (myself) to teach version control concepts. The project demonstrates fundamental Git concepts, including local version control and GitHub repository management.

### Purpose

This project was designed as an educational resource for learning:
- **Local Version Control**: Understanding how Git tracks changes locally
- **GitHub Repository Management**: Learning how to work with remote repositories
- **Basic Git Workflows**: Exploring fundamental Git commands and operations

### Contents

This project contains simple files to demonstrate basic Git usage and does not offer any complex or production-ready features. It serves purely as a learning tool for Git fundamentals.
